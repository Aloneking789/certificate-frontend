# **App Name**: Eunous CertiFlow

## Core Features:

- Public Verification Portal: A public search engine where stakeholders can verify credentials instantly using unique registration numbers or QR codes.
- Administrative Command Center: A secure dashboard providing real-time metrics on certificate issuance and verification trends.
- Intelligent Generation Engine: An automated form for batch or individual certificate creation with dynamic registration number allocation.
- AI Credential Audit Tool: An AI tool that reasons through certificate metadata to cross-reference reported internship hours against domain-standard curriculum to flag inconsistencies.
- Instant QR Signature Integration: Cryptographically secure QR codes generated automatically for every certificate, allowing direct mobile verification.
- PDF Perfectionist Render: High-fidelity PDF generation using jspdf and react-to-print for professional, frame-ready documentation.
- Cloud-Synched Asset Management: Secure storage and retrieval of all internship metadata through a dedicated API integration.

## Style Guidelines:

- Primary Color: Royal Trust Blue (#1E3A8A), used for key brand elements and headers to evoke corporate reliability.
- Background: Sky Wash (#F1F4F9), a desaturated blue-tinted white that maintains the corporate theme while ensuring readability.
- Accent Color: High-Action Cyan (#0EA5E9), used sparingly for primary buttons and verification badges to drive user engagement.
- Headline Font: 'Space Grotesk' (Sans-serif) for a modern, tech-forward look; Body Font: 'Inter' (Sans-serif) for high legibility across density-rich tables.
- Clean, thin-line geometric icons that align with a minimal aesthetic, focusing on clarity over decoration.
- A sophisticated split-panel layout for administrative tasks and a high-impact, centered focus for public certificate views.
- Progressive loading transitions and micro-interactions on the verification badge to provide instant feedback to the user.
